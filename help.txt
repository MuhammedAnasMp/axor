# Axon CLI Tools Help

## Web & OTA Updates (from frontend folder)
npm run build-ui          - Build frontend ZIP + register in databases (local/prod) in testing mode
npm run build-ui push     - Build ZIP + register + git commit + push to GitHub
npm run build-ui publish  - Approve current testing version to all users
npm run build-ui remove   - Remove current testing OTA version

## Build Standalone Applications (from frontend folder)
npm run aapk / apk        - Compile Android debug APK (.apk)
npm run exe               - Compile Windows desktop application (.exe)
npm run both              - Compile both Android APK and Windows EXE

## Local Development (from frontend folder)
npm run dev               - Run Vite local development server
npm run electron:start    - Run Electron app shell locally (pointing to dev server or dist)
